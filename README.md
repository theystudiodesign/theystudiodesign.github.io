# THE'Y STUDIO DESIGN — Gestion

Application de gestion des clients, projets, tâches et paiements — **THE'Y STUDIO DESIGN**.

## 🚀 Kifach t-publiiha 3la GitHub Pages (5 minutes)

1. Sawb compte f [github.com](https://github.com) (majani)
2. Clicki **+** (lfo9 3la limen) → **New repository**
   - Repository name: `they-gestion`
   - Khtar **Public**
   - Clicki **Create repository**
3. F la page dyal le repo → clicki **uploading an existing file**
   - Jbed (drag & drop) ga3 les fichiers dyal had le dossier: `index.html`, `manifest.webmanifest`, `sw.js`, `icon-192.png`, `icon-512.png`
   - Clicki **Commit changes**
4. Sir l **Settings** → **Pages** (f la liste 3la lisser)
   - F **Branch**: khtar `main` o `/ (root)` → **Save**
5. Tsenna 1-2 minutes → l'app dyalek ghadi tkoun f:
   **`https://<smiytk-f-github>.github.io/they-gestion/`**

## 💻 Kifach t-installiha f PC b7al application

1. 7ell le lien dyalek f **Chrome** wla **Edge**
2. Clicki 3la l'icône dyal **Install** (⊕ f la barre dyal l'adresse) wla Menu ⋮ → **Install THE'Y Gestion**
3. Safi! Ghadi tkoun 3andek icône f le bureau, kat7ell f fenêtre dyalha bou7dha, o khedama **offline**

## ⚠️ Mohim — les données

- Les données kayt-sauvegardaw f **le navigateur** (localStorage) dyal koula appareil
- Dir **Export JSON** (f sidebar) mn temps l'autre bach ikoun 3andek backup
- Bach tn9el les données mn PC l akhor: Export JSON f lewwel → Import JSON f tani

Clicki 3la blasst le logo f sidebar → khtar le PNG dyal logo dyalek (version bida) → kayb9a msjjel.
